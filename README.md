## 🏗️ ARQUITETURA TÉCNICA

### **📁 ESTRUTURA DO PROJETO:**
```
discovery-notecraft/
├── 📂 src/app/                    # Next.js 14 App Router
│   ├── 📂 api/                    # Serverless Functions
│   │   ├── analyze-discovery.ts   # 🧠 Agente IA Analisador
│   │   └── export-pdf.ts         # 📄 Gerador PDF
│   ├── 📂 dashboard/              # 📊 Interface Principal
│   ├── 📂 relatorio/[id]/         # 📋 Página de Relatório
│   └── 📂 components/             # 🧩 Componentes UI
├── 📂 lib/                       # 🔧 Utilities
│   ├── supabase.ts               # 🗄️ Cliente Supabase
│   ├── ai-engine.ts              # 🤖 Engine IA
│   └── data-schemas.ts           # 📋 Schemas TypeScript
├── 📂 database/                  # 🗄️ Schema Supabase
└── 📂 docs/                      # 📚 Documentação
```

### **🗄️ BANCO DE DADOS (Supabase):**
- `form_submissions` - Dados do questionário
- `discovery_analysis` - Análises do agente IA  
- `section_insights` - Insights por seção
- `agent_recommendations` - Agentes recomendados
- `package_configurations` - Configuração de pacotes

### **🤖 STACK TECNOLÓGICO:**
- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **Backend:** Vercel Serverless Functions
- **Database:** Supabase (PostgreSQL)
- **IA:** OpenAI GPT-4o / Claude 3.5 Sonnet
- **Charts:** Recharts
- **PDF:** React-PDF
- **Deploy:** Vercel
